# ClientCRUD

Sistema CRUD de gerenciamento de clientes desenvolvido em Python com SQLite.

##  Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Busca por ID
- Atualização de dados
- Exclusão de clientes

##  Tecnologias utilizadas

- Python
- SQLite
